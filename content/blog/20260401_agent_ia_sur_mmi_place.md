# Arrivée d'un agent IA sur MMI Place

Pour améliorer votre quotidien, nous voyons les choses en grand ! Dès la semaine prochaine, MMI Place sera équipé d'un agent IA propulsé par Microsoft Copilot afin de répondre à toutes vos questions. En liant votre compte Github à votre compte MMI Place, vous pourrez également bénéficier d'un suivi entre VSCode et MMI Place, sauf si vous désactivez la fonctionnalité sur votre compte Microsoft.

## Les avantages pour vous

Cet agent IA intégré vous permettra de:

- **Obtenir des réponses instantanées** à vos questions académiques et techniques
- **Explorer les cours disponibles** sans quitter la plateforme
- **Gérer vos événements et plannings** avec l'aide d'une IA intelligente
- **Optimiser votre workflow** en intégrant Copilot directement dans VSCode via l'extension Github Copilot

La synchronisation automatique avec votre compte Microsoft assure une expérience fluide et personnalisée, adaptée à votre progression académique.

## Comment ça marche ?

Vous avez juste à aller sur la barre de recherche en haut du site MMI Place et d'y entrer le code `Poisson d'avril 🐠`. Vous devrez ensuite envoyer cet article à l'un de vos camarades pour qu'il tombe lui aussi dans le piège
